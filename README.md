# 金河分镜

影视分镜与制片管理工具，桌面端（macOS / Windows / Linux）。源码闭源，本仓用于发布安装包与收集反馈。

## 下载

到 [Releases](https://github.com/JinHe9527/jinhe-storyboard-releases/releases) 选对应平台安装包：
- macOS：`.dmg`
- Windows：`.exe`
- Linux：`.AppImage`
